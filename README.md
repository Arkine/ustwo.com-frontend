# ustwo.com-prototype

  * Building using Gulp
  * CSS stolen from live ustwo site
  * Routing using Page.js
  * Templating using Handlebars.js
  * Animations via GreenSock.js

To update content, have a look in `/data` folder!

## Setup

    $ git submodule init && git submodule update
    $ bundle install
    $ npm install
    $ npm run bower

## Develop

    $ npm run watch
